captcp template(1)
====================

NAME
----
captcp template - provides a wide varity of Gnuplot template files

SYNOPSIS
--------
[verse]
'captcp' template [--verbose <debug | info | error >] [--list] <templatename>

DESCRIPTION
-----------
Provides useful Gnuplot and Makefile templates for visualization. Also used
internal.
