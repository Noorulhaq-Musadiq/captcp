captcp(1)
=======

NAME
----
captcp - TCP Analysis Tool Collection for Linux

SYNOPSIS
--------
[verse]
'perf' [--help] COMMAND [ARGS]

Command is one off the following:

* stacktrace (see linkcaptcp:captcp-stacktrace[1])
* show (see linkcaptcp:captcp-stacktrace[1])
* statistic (see linkcaptcp:captcp-stacktrace[1])
* payloadtimeport (see linkcaptcp:captcp-stacktrace[1])
* timesequence (see linkcaptcp:captcp-stacktrace[1])
* geoip (see linkcaptcp:captcp-stacktrace[1])
* throughtput (see linkcaptcp:captcp-stacktrace[1])
* connection (see linkcaptcp:captcp-stacktrace[1]) 
* template (see linkcaptcp:captcp-stacktrace[1])
* sequencegraph (see linkcaptcp:captcp-stacktrace[1])

DESCRIPTION
-----------
Captcp collect a number of Network Analysis tools. Focus is TCP but there are
some general purpose modules as well.
